### Thing or Two Take Home Test

#### Installation

- Clone the repo

```
git clone https://github.com/lexarisa/csv_file_reader.git

```

- Install NPM packages

```
npm install

```

- Seed the database

```
cd server
npx prisma db seed

```

- Run the app
```
cd client
npm run start
```

- Run the server
```
cd server
npm run start
```
